CHIP-8 Emulator in Go, Ben Coleman
GitHub Project
Controls:
Note. Each game uses different keypad inputs, sorry but you will have to experiment, but WASD is common for movement

  • F5 - Pause the system
  • F6 - When paused step through instructions (useful when debug is enabled)
  • F11 - Enable debugger, which will be output to stdout or the browser console
  • F12 - Reset system and reload the program
  • [ - Slow down the emulation
  • ] - Speed up the emulation
